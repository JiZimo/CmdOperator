简介

·简介

CmdOperator是一款非常简洁易用的程序库，它用于实现各种命令行程序的交互部分

·如何使用

若需要在您的程序中使用该程序库，只需将在此项目根文件夹下的文件CmdOperator.h拷贝到您的工作目录中，然后添加

#include "CmdOperator.h"

即可使用该程序库

创建CmdOperator对象

·构造函数

有两种构造函数可以使用

CmdOperator()

介绍：可以构造一个默认的CmdOperator对象

详见

CmdOperator(const std::string &_op, const std::function<int(const std::string&)> _eh = nullptr, const std::map<std::string, commandHandler> _sheet = {})

介绍：在该函数中必须提供命令提示符，_sheet是一个命令与处理函数对应关系的注册表（以下简称注册表），默认为空，eh为错误处理函数，即，当注册表中没有与用户输入的项目对应时会调用错误处理函数，并传递该无法对应的命令

详见

注意：CmdOperator类不提供拷贝构造函数和赋值运算符，所以你不能创造某一CmdOperator对象的拷贝

成员函数操作

·命令操作

-添加项目操作

commandHandler addCommand(const std::string &str, const commadnHandler func)

介绍：注册一条命令，在该命令被用户使用时，将调用func，其中commandHandle为函数类型，该函数接受一个CmdOperator类的指针为当前命令行，以及一个只读的std::list<std::string>类型，该链表中储存着用户提供的参数，如，当用户提供 command arg1 arg2是，对应着command的注册表项中的函数的第二个参数就会被赋值为{"arg1","arg2"}

详见

commandHandler addCommand(const std::pair<const std::string, const commandHandler>

介绍：同上，不过将两个参数整合为一个std::pair

详见

std::map<std::string, const commanadHandler> addCommand(const std::map<const std::string, const commandHandler> list

介绍：注册list中的项目，返回该list

详见

-更改项目操作

commandHandler replaceCommand(const std::string &index, const commandHandler newFunc)

介绍：更改由index指定的注册项为newFunc

详见

commandHandler replaceCommand(const std::pair<const std::string, const commandHandler> p)

介绍：同上，不过以std::pair传递参数
