commandHandler addCommand(const std::string &str, const commadnHandler func)

描述：注册一个命令，当该命令被用户使用时会将命令后的参数以链表（std::list<std::string>）的形式传入func来调用func

异常

该函数发现提供的str已注册时，会抛出std::invalid_argument

返回值

该函数返回被注册的函数

示例

CmdOperator cmd;

CmdOperator::commandHandler func=...;

cmd.addCommand("command",func);

commandHandler addCommand(const std::pair<const std::string, const commandHandler> p)

描述：等同于上面的函数，只不过以std::pair类型来传递参数

异常

该函数会在p.first已被注册时抛出std::invalid_argument

返回值

该函数返回提供的pair

示例

    
    CmdOperator cmd;
    CmdOperator::commandHandler func=...;
    cmd.addCommand(std::make_pair("command",func));
std::map<std::string, const commanadHandler> addCommand(const std::map<const std::string, const commandHandler> list

描述：注册list中的每一项

异常

该函数会在任意一项已被注册时抛出std::invalid_argument

返回值

该函数返回用户提供的列表list

示例

    
    std::map<std::string, const commanadHandler> list={...}
    CmdOperator cmd;
    cmd.addCommand(list);
