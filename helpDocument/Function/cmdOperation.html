int exitCommandLine()

描述：退出命令行

异常

该函数不会抛出异常

返回值

返回__result值

示例

    
        //下面是一般的退出命令行函数实现方法
        auto exit=[](CmdOperator *cmd,std::list<std::string> argList){
            return cmd->exitCommandLine;
        };
        //然后可以将此函数注册为退出指令
        //如：
        //aCmdOperatorObject.addCommand("exit",exit);
int execute(std::ostream &out,std::istream &in)

描述：执行CmdOperator，并以out为输出流，in为输入流

异常

如果注册的函数或错误处理函数中可能抛出异常，则该函数就会抛出异常

返回值

返回__result值

示例

        
            #include <iostream>
            CmdOperator cmd;
            //注册一些函数
            //...
            cmd.execute(std::cout,std::cin);
