commandHandler replaceCommand(const std::string &index, const commandHandler newFunc)

描述：将index所对应注册的函数更改为newFunc

异常

若index指定的索引不存在，则抛出std::invalid_argument异常

返回值

该函数返回用户提供的函数

示例

    
        CmdOperator::commandHandler aFunc=...;
        replaceCommand("index",aFunc);
commandHandler replaceCommand(const std::pair<const std::string, const commandHandler> p)

描述：同上，不过以std::pair传递参数

异常

若给定的std::pair的第一项不存在，则抛出std::invalid_argument异常

返回值

该函数返回用户提供的std::pair

示例

    
        CmdOperator::commandHandler aFunc=...;
        replaceCommand(std::make_pair("index",aFunc));
std::map<const std::string, const commandHandler> replaceCommand(const std::map<const std::string, const commandHandler> list)

描述：逐个的改变list中被注册的命令

异常

若其中存在索引不存在的情况，则抛出std::invalid_argument异常

返回值

返回用户提供的列表list

示例

    
        std::map<const std::string,const commandHandler> list={...};
        replaceCommand(list);
