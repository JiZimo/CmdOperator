CmdOperator构造函数手册页

CmdOperator()

描述：该函数是CmdOperator的默认构造函数，该函数将构造一个以">"为提示符的命令行系统

示例

CmdOperator cmd(); CmdOperator(const std::string &_op, const std::function<int (const std::string&)> _eh = nullptr, const std::map<std::string, commandHandler> _sheet = {})

描述：该构造函数要求用户提供要构造的命令行系统的命令提示符。_eh为可选项，该项为被构造的命令行系统的错误处理函数，即，当用户输入的命令与已注册项不能成功匹配时，调用该函数，并传入该无法匹配的命令。_sheet提供将被命令行系统注册的命令，默认为空，即不提供。

示例

CmdOperator cmd1("_>");

或

CmdOperator cmd2("_>",[](const std::string &command)->int{std::cout<<command<<" invalid"<<std::endl; return 0;},{{"command",aFunction}});
